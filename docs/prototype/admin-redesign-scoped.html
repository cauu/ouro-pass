7 · 通用 Table 范式（当前范围）

统一表格外观：表头大写小字、行悬停高亮、状态用 badge、hash/kid 等以等宽并可点击复制、行操作悬停显隐、表尾计数。当前 API 一次性返回全量列表，故本版不含搜索 / 排序 / 分页 / 批量。

已引入但未使用的 @tanstack/react-table 可在后端补齐分页 / 过滤参数后，抽成统一 <DataTable>，承载搜索 / 排序 / 分页 / 行选 —— 详见北极星版 admin-redesign.html。

8 · 反馈：Toast / 弹窗 / 抽屉

轻量结果用 Toast；危险/需确认用居中弹窗（敏感操作内嵌 step-up 签名）；多字段创建用右侧抽屉。统一替换代码中的原生 prompt()/confirm()（见 Channels 的换 Token / 删除）。
